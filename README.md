# Playwright Project for AIAgentMCP

This project is set up for end-to-end testing using Playwright.

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Run all tests:
   ```sh
   npx playwright test
   ```
3. Open Playwright Test UI:
   ```sh
   npx playwright test --ui
   ```

## Project Structure
- `tests/` - Example Playwright tests
- `playwright.config.ts` - Playwright configuration

## Documentation
See [Playwright documentation](https://playwright.dev/) for more details.
